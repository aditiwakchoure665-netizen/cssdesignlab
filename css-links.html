CSS Links

CSS links are used to style hyperlinks in a web page. Links can be styled with different colors, fonts, backgrounds, and hover effects.

Link States

CSS provides four link states:

  • a:link – normal unvisited link
  • a:visited – visited link
  • a:hover – when the mouse is over the link
  • a:active – when the link is clicked

Example

a:link {
color: blue;
}

a:visited {
color: purple;
}

a:hover {
color: red;
}

a:active {
color: green;
}
