CSS Navigation Bar

A navigation bar is used to help users move between different pages of a website. It usually contains links such as Home, About, Tutorials, and Contact.

Example CSS for Navigation Bar

nav ul{
list-style:none;
margin:0;
padding:0;
}

nav li{
float:left;
}

nav li a{
display:block;
padding:14px;
background:#333;
color:white;
}

CSS is used to style the navigation bar, change colors, align links, and add hover effects. Navigation bars are an important part of modern website design.
