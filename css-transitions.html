CSS Transitions

CSS transitions allow you to change property values smoothly over a given duration. They help create animations when a property changes from one value to another.

Basic Syntax

selector {
transition: property duration;
}

Example

.box{
background: blue;
transition: background 1s;
}

.box:hover{
background: red;
}
