CSS Syntax

CSS syntax is used to apply styles to HTML elements. A CSS rule consists of a selector and a declaration block.

Basic Structure of CSS

selector {
property: value;
}

Example

h1 {
color: blue;
font-size: 30px;
}

In the example above:

  • h1 is the selector.
  • color is the property.
  • blue is the value.

The declaration block contains one or more property-value pairs separated by semicolons.
