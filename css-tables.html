CSS Tables

CSS is used to improve the appearance of HTML tables by adding borders, colors, spacing and alignment.

Example Table

Property Description
border Adds borders to table elements
padding Adds space inside table cells
text-align Aligns text inside table cells
background-color Adds background color to rows

Example CSS Code

table {
width:100%;
border-collapse:collapse;
}

th, td {
border:1px solid black;
padding:10px;
text-align:center;
}

th {
background-color: black;
color: white;
}
