CSS Colors

Learn how to apply colors in CSS

What are CSS Colors?

CSS colors are used to set the color of text, backgrounds, borders, and other elements on a webpage. Colors help improve the design and visual appearance of a website.

1. Color Names

You can use predefined color names in CSS.


p {
    color: red;
}

Example colors: red, blue, green, yellow, black, white.

2. HEX Colors

HEX values represent colors using hexadecimal numbers.


h1 {
    color: #3498db;
}

HEX colors start with # followed by six characters.

3. RGB Colors

RGB stands for Red, Green, and Blue values.


div {
    background-color: rgb(255, 0, 0);
}

The values range from 0 to 255.

4. RGBA Colors

RGBA includes transparency (alpha value).


div {
    background-color: rgba(0, 0, 255, 0.5);
}

The alpha value ranges from 0 (transparent) to 1 (solid).

5. HSL Colors

HSL stands for Hue, Saturation, and Lightness.


p {
    color: hsl(120, 100%, 50%);
}

This method allows easier color adjustments.

Example


body {
    background-color: #f0f8ff;
}

h1 {
    color: rgb(0, 102, 204);
}

p {
    color: green;
}

Summary

  • CSS colors improve webpage design
  • Colors can be applied to text, background, and borders
  • Common formats include Color Names, HEX, RGB, RGBA, and HSL
