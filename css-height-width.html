CSS Height and Width

The CSS height and width properties are used to set the height and width of an element. These properties help control the size of elements such as images, divs, and sections.

Example CSS Code

div {
width: 300px;
height: 150px;
background-color: lightblue;
}

Example Output

300px Width & 150px Height

In this example, the width of the box is 300 pixels and the height is 150 pixels.
