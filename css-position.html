CSS Position

The CSS position property is used to specify how an element is positioned on a web page. It allows elements to be placed in different locations on the screen.

Types of CSS Position

  • Static – Default positioning of elements.
  • Relative – Positioned relative to its normal position.
  • Absolute – Positioned relative to the nearest positioned ancestor.
  • Fixed – Positioned relative to the browser window.
  • Sticky – Toggles between relative and fixed positioning.

Example

div {
position: relative;
left: 20px;
top: 10px;
}

In this example, the element moves 20 pixels from the left and 10 pixels from the top relative to its normal position.
