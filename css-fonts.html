CSS Fonts

CSS fonts are used to control the appearance of text on a webpage. Using CSS, you can change the font family, size, style, and weight of text.

Font Properties in CSS

  • font-family
  • font-size
  • font-style
  • font-weight

Example

p {
font-family: Arial;
font-size: 18px;
font-style: italic;
font-weight: bold;
}

In the example above:

  • font-family defines the type of font.
  • font-size controls the size of the text.
  • font-style makes the text italic or normal.
  • font-weight controls how bold the text appears.

Using CSS fonts helps improve the readability and design of a website.
