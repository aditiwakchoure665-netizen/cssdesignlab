CSS Selectors

Learn how to select HTML elements using CSS

What are CSS Selectors?

CSS selectors are used to select and style HTML elements. They define which HTML elements the CSS rules will apply to.

1. Element Selector

The element selector selects HTML elements by their name.


p {
    color: blue;
}

This will change the color of all paragraph elements.

2. ID Selector

The ID selector selects an element with a specific ID.


#header {
    background-color: black;
    color: white;
}

It is used with the # symbol.

3. Class Selector

The class selector selects elements with a specific class.


.box {
    border: 2px solid gray;
    padding: 10px;
}

It is used with the . symbol.

4. Group Selector

The group selector applies the same style to multiple elements.


h1, h2, p {
    font-family: Arial;
}

This will apply the same font style to all h1, h2, and paragraph elements.

5. Universal Selector

The universal selector selects all HTML elements.


* {
    margin: 0;
    padding: 0;
}

This is commonly used for resetting default browser styles.

Summary

  • Selectors are used to target HTML elements
  • They allow developers to apply styles efficiently
  • Common selectors include element, id, class, group, and universal selectors
