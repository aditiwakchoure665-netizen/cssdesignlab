CSS Box Model

The CSS Box Model is a fundamental concept used to design and layout web pages. Every HTML element is considered as a rectangular box.

Components of the Box Model

  • Content – The actual content like text or images.
  • Padding – Space between content and border.
  • Border – Surrounds the padding and content.
  • Margin – Space outside the border.

Example

div {
width: 300px;
padding: 20px;
border: 5px solid black;
margin: 30px;
}

In this example, the width of the content is 300px, padding adds space inside the box, the border surrounds the box, and margin creates space outside the box.
