CSS Text

CSS Text properties are used to style and format text on web pages. These properties control alignment, decoration, spacing, and transformation of text.

Common CSS Text Properties

  • text-align
  • text-decoration
  • text-transform
  • text-indent
  • letter-spacing
  • line-height

Example

p {
text-align: center;
text-transform: uppercase;
letter-spacing: 2px;
}

In this example:

  • text-align aligns the text.
  • text-transform changes the case of letters.
  • letter-spacing controls space between letters.
