CSS Padding

Padding is used to create space inside an element, between the content and the border. It helps improve the layout and readability of web pages.

Example of CSS Padding

div {
padding: 20px;
}

This adds 20 pixels of space inside the element.

Padding Example Output

This box has 20px padding inside the border.

Different Padding Properties

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Example

div{
padding-top:10px;
padding-right:20px;
padding-bottom:10px;
padding-left:20px;
}
