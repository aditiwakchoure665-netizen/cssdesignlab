CSS Display Property

The CSS display property defines how an HTML element is displayed on a web page. It controls whether an element appears as a block, inline, or other layout types.

Common Display Values

  • block – The element takes the full width of the page.
  • inline – The element only takes the required width.
  • inline-block – Similar to inline but allows width and height.
  • none – The element is hidden and not displayed.

Example

div {
display: block;
}

span {
display: inline;
}

In the example above, the div element will behave like a block element, while the span element will behave like an inline element.
