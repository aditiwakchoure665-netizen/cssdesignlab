CSS Margins

CSS margins are used to create space around elements outside their borders. Margins help improve the layout and readability of a webpage.

Margin Properties

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Example

div {
margin: 20px;
}

The code above adds 20 pixels of space around the element.

Example Box

This box has margin around it.

Margins can also be set individually: 

div {
margin-top: 10px;
margin-right: 20px;
margin-bottom: 10px;
margin-left: 20px;
}
