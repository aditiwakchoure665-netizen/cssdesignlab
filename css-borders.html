CSS Borders

Learn how to create borders using CSS

What are CSS Borders?

CSS borders are used to define the outline around an HTML element. Borders can have different styles, colors, and widths to enhance the visual design of web pages.

1. Border Style

The border-style property defines the style of the border.


p {
    border-style: solid;
}

Common border styles:

  • solid
  • dotted
  • dashed
  • double
  • groove
  • ridge
  • inset
  • outset

2. Border Width

The border-width property sets the thickness of the border.


p {
    border-style: solid;
    border-width: 3px;
}

3. Border Color

The border-color property sets the color of the border.


p {
    border-style: solid;
    border-color: red;
}

4. Border Shorthand

You can combine width, style, and color in a single property.


p {
    border: 2px solid blue;
}

5. Rounded Borders

The border-radius property creates rounded corners.


div {
    border: 2px solid black;
    border-radius: 10px;
}

Example


.box {
    border: 3px dashed green;
    padding: 10px;
}
This is an example of a bordered box.

Summary

  • CSS borders define the outline of elements
  • You can control style, width, and color
  • Rounded borders can be created using border-radius
