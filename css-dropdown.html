CSS Dropdown Menu

A dropdown menu is a list of options that appears when the user hovers or clicks on a button. CSS can be used to create dropdown menus without JavaScript.

Example of CSS Dropdown

HTML CSS JavaScript

CSS Code Used

.dropdown:hover .dropdown-content{
display:block;
}

When the user moves the mouse over the dropdown button, the menu items appear.
