CSS Clear Property

The CSS clear property is used to control the behavior of floating elements. It specifies which sides of floating elements are not allowed to float beside the cleared element.

Values of Clear Property

  • none – Default value. Allows floating elements on both sides.
  • left – No floating elements allowed on the left side.
  • right – No floating elements allowed on the right side.
  • both – No floating elements allowed on either side.

Example

img {
float: left;
}

p {
clear: left;
}

In this example, the paragraph will move below the floating image instead of appearing beside it.
