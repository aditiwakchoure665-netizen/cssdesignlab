CSS Float

The CSS float property is used to position an element to the left or right of its container. Other content can flow around the floated element.

Example

This box is floated to the left using CSS float.

This paragraph flows around the floated element. Float is commonly used for creating layouts and positioning images with text.

CSS Code Example

img {
float: right;
}

div {
float: left;
}

Possible values for the float property include:

  • left
  • right
  • none
  • inherit
