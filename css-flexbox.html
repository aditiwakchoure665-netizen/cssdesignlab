CSS Flexbox

Flexbox (Flexible Box Layout) is a CSS layout module used to design flexible and responsive layouts. It allows elements to align and distribute space within a container.

Basic Flexbox Example

.container{
display:flex;
}

Example Layout

Box 2
Box 3

In this example, the container uses display: flex which arranges the boxes in a row.

Common Flexbox Properties

  • display: flex
  • flex-direction
  • justify-content
  • align-items
  • flex-wrap
