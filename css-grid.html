CSS Grid

CSS Grid Layout is a powerful layout system used to create two-dimensional layouts for web pages. It allows developers to arrange elements into rows and columns easily.

Basic CSS Grid Syntax

.container {
display: grid;
grid-template-columns: 1fr 1fr 1fr;
gap: 10px;
}
