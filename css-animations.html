CSS Animations

CSS animations allow elements on a web page to gradually change from one style to another. Animations can change properties like color, position, size, and many other visual effects.

Basic Animation Syntax

@keyframes animation-name {
from {property: value;}
to {property: value;}
}

selector {
animation-name: animation-name;
animation-duration: 5s;
}

Example

The box below moves from left to right using CSS animation.

In this example:

  • @keyframes defines the animation.
  • animation property applies the animation to the element.
  • 5s means the animation lasts for 5 seconds.
